CSS Documentation – Indy Eats Final Project

CSS Rule Explanation
* { box-sizing: border-box; } Keeps padding and borders from messing up sizing — makes layout way easier to manage.
body { font-family: Arial; } Makes sure the text looks clean and easy to read across the whole site.
nav ul { display: flex; justify-content: center; } Uses Flexbox to center the nav links across the top.
.flip-card, .flip-card-inner Lets the flip cards spin around when you hover — it’s all done with transform and 3D styles.
.grid-layout { display: grid; grid-template-columns... } Lays out articles in a grid that adjusts depending on screen size.
@media screen and (max-width: 600px) Tweaks the layout for phones — makes nav stack and flip cards go vertical.
footer { background-color, padding } Gives the footer matching colors and padding so it looks clean and consistent.
.flip-card-back a { background-color, border-radius } Makes the flip card buttons look clickable and styled — rounded and colored.
img { max-width: 100%; height: auto } Keeps images from overflowing — they resize nicely in their containers.
